Esse site vai lhe permitir multiplicar duas matrizes 3x3 entre elas. No programa usado para construir o site o que seria linha 1 e coluna um vai ser definido como linha 0 e coluna 0, o que seria linha 2 e coluna 2 vai ser definido como linha 1 e coluna 1, assim em diante usando a mesma lógica para as seguintes linhas e colunas.

O cálculo entre matrizes é feito multiplicando linha por coluna.

Clique no botão a seguir para indicar os números e realizar o cálculo.
